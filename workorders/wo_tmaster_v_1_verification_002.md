## 4. TASK DESCRIPTION

For each `tmaster*.html` file, perform the following checks:

### 4.1 Structural Validation

- Header section present
- Reviewer information section present
- Content form present containing:
  - Exactly **45 chapter title blocks**
  - Exactly **351 paragraph blocks**
- Footer section present

### 4.2 Required PR Elements

Verify presence of all `required_pr_elements` defined in:

```
presentation.json → container_types.tmaster.validation.required_pr_elements
```

### 4.3 Required Data Attributes

Verify required data attributes are present where specified:

- `data-pr-id`
- `data-pr-type`
- `data-id`
- `data-chapter`
- `data-original`

(Matches `presentation.json` exactly)

### 4.4 Required Features

Verify presence of the following features:

- Language toggle functionality
- Modification tracking
- Submit functionality

### 4.5 Language Binding Validation

For each file, confirm:

- Correct `pr_title` binding for the target language
- Correct `pr_edit` binding for the target language
- Correct master file reference as defined in `presentation.json → language_config`
